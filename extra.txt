// TryStatement
TryStatement: try Block Catches
| try Block Catches Finally
| try Block  Finally
| TryWithResourcesStatement
;



Catches: CatchClauseList

CatchClauseList :  CatchClause | CatchClauseList CatchClause


CatchClause: catch LeftParenthesis CatchFormalParameter RightParenthesis 

CatchFormalParameter: CatchType VariableDeclaratorId | VariableModifierList CatchType VariableDeclaratorId

CatchType: UnannClassType | UnannClassType BitOrClassTypeList


TryWithResourcesStatement: try ResourceSpecification Block  
| try ResourceSpecification Block Catches 
| try ResourceSpecification Block  Finally
| try ResourceSpecification Block Catches Finally

ResourceSpecification: LeftParenthesis ResourceList Semicolon RightParenthesis
| LeftParenthesis ResourceList Semicolon RightParenthesis

ResourceList: Resource | Resource SemicolonResourceList

SemicolonResourceList : Semicolon Resource | SemicolonResourceList Semicolon Resource

Resource: LocalVariableDeclaration
| VariableAccess

VariableAccess: ExpressionName
| FieldAccess
